# Tai AI Automation Landing Page

Premium conversion landing page for `nguyenvantai.com`.

## Stack

- Next.js 16 App Router
- React 19 + TypeScript strict
- Tailwind CSS v4
- shadcn/ui primitives
- Framer Motion
- Space Grotesk display font
- Be Vietnam Pro body font

## Current Scope

Phase 02 is complete: content copy, owner story, privacy policy, Terms of Service, footer/contact lock, and legal content are done. Phase 03 next.

Key routes:

- `/` - Aurora layout preview
- `/brand` - owner photo + favicon verification
- `/logo` - production logo verification
- `/components` - shadcn/Aurora component verification
- `/animations` - motion pattern verification

## Commands

```bash
pnpm dev
pnpm lint
pnpm build
```

On Windows PowerShell with script policy restrictions, use:

```bash
pnpm.cmd lint
pnpm.cmd build
```
